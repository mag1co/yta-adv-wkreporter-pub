1. Install the app

Install from the YouTrack Marketplace or upload the ZIP file manually to your YouTrack instance (YouTrack Admin → Apps).

2. Attach to a dashboard or project

Go to your Dashboard, click Add Widget, and select Adv.Weekly Reporter. Alternatively, enable it as a Project tab.

3. Configure your report

Click the Edit button on the widget to open its settings:

  • Widget Title: (Optional) Give your widget a custom name.
  • YouTrack Query: Enter your search query (e.g., #Unresolved project: Design).
  • Project Context: Choose a project to load its specific custom fields. Auto-detects if you specify a project in the query above!
  • Columns to Display: Choose the exact fields to show (e.g., summary, description, Assignee, State, comment).
  • Periods to Show: Select how many historical weeks of data to group and display.

4. Export to Excel

Expand any weekly period and click the Export to XLSX button to instantly download a spreadsheet containing all issues, complete with clickable links and the latest comments.
